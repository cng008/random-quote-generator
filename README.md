# Random Quote Machine

This app is a randomly generated quote machine that allows users to get new quotes and share them on Twitter.

## API

## Tech Stack

- Axios
- Bootstrap
- Font Awesome
- React
- Redux

### How to Run

```bash
git clone https://github.com/cng008/spotify-web-player-clone.git
npm i
npm start
```
